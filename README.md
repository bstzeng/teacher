# 慢慢學

一個可持續擴充主題與課程的純靜態教學網站。

## 目前課程

- 飛機與機場 01：桃園機場常見客機（10 種機型）
- 飛機與機場 02：A320 駕駛艙互動探險
- 飛機與機場 03：Airbus A320 vs Boeing 737 駕駛艙比較

## 技術

- HTML
- CSS
- JavaScript
- GitHub Pages

## 本機預覽

直接開啟 `index.html`，或在專案目錄啟動任意靜態檔案伺服器。

```bash
python3 -m http.server 8000
```

接著開啟 <http://localhost:8000>。

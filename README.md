# 慢慢學

一個可持續擴充主題與課程的純靜態教學網站。

首頁採「主題 → 課程」兩層架構；目前已建立航空主題。

## 目前課程

- 飛機與機場 01：桃園機場常見客機（10 種機型）
- 飛機與機場 02：A320 駕駛艙互動探險
- 飛機與機場 03：Airbus A320 vs Boeing 737 駕駛艙比較
- 飛機與機場 04：客機硬體構造互動解剖
- 飛機與機場 05：桃園 LOC RWY 05L 進場航圖互動教學
- 飛機與機場 06：飛行四力、迎角與失速互動實驗
- 飛機與機場 07：三軸控制與協調轉彎互動教學
- 飛機與機場 08：從登機門到登機門的完整航班流程
- 飛機與機場 09：跑道編號、L/C/R 與跑道標線解說

## 本機預覽

直接開啟 `index.html`，或在專案目錄啟動任意靜態檔案伺服器。

```bash
python3 -m http.server 8000
```

接著開啟 <http://localhost:8000>。
